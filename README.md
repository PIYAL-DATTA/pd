## ✨ Features

- **WebGL Fluid Simulation Hero** — Interactive canvas-based fluid simulation as a full-screen hero background
- **Animated Background Blobs** — Ambient blob animations for a layered depth effect
- **Custom Cursor** — Glow dot + ring cursor that follows the mouse
- **Scroll Progress Bar** — Thin accent bar at the top tracking scroll position
- **Typed.js Subtitle** — Cycling role titles in the hero section
- **AOS Scroll Animations** — Fade-in animations triggered on scroll throughout the page
- **Responsive Design** — Fully mobile-friendly layout using Bootstrap 5 grid
- **Sticky Navbar** — Glassmorphism navbar with smooth scroll links
- **Tabbed Resume Section** — Education, Experience & Interests via Bootstrap Pills
- **Timeline Cards** — Styled cards for education and work experience
- **Skills Grid** — Categorized skill badges (Frontend, Backend, Languages)
- **Project Showcase** — 8 project cards with tech stacks and feature lists
- **Marquee References** — Infinite-scrolling forward/backward reference ticker
- **Contact Section** — CTA cards (Location, Phone, Email) + Social buttons + Embedded Google Map
- **Downloadable CV** — Direct PDF download button in hero and about sections
- **SEO Ready** — Meta tags, Open Graph, and mobile-specific meta configured

## 🚀 Getting Started

This is a **static website** — no build step or server required.

1. **Clone the repository**
   ```bash
   git clone https://github.com/PIYAL-DATTA/PIYAL-DATTA.github.io.git
   ```

2. **Open locally**
   - Just open `index.html` in any modern browser.
   - Or use a local server (e.g., VS Code **Live Server** extension) for the best experience.

3. **Deploy**
   - Hosted automatically via **GitHub Pages** at [piyal-datta.github.io](https://piyal-datta.github.io/)
